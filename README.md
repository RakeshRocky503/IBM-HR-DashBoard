# IBM HR Analytics Dashboard

## 📌 Project Overview

The IBM HR Analytics Dashboard is an interactive dashboard built using **Microsoft Excel**. It helps analyze employee data and provides valuable insights into workforce trends, employee attrition, and HR performance.

The dashboard enables users to explore HR data through interactive charts, KPI cards, Pivot Tables, Pivot Charts, and Slicers, making it easier to identify patterns and support data-driven decision-making.

## 🎯 Objectives

* Analyze employee attrition trends.
* Monitor key HR metrics.
* Understand employee demographics.
* Identify factors affecting employee retention.
* Provide an interactive and user-friendly HR dashboard.

## 📊 Key KPIs

* Total Employees
* Total Employees Left
* Average Attrition Percentage
* Average Monthly Income
* Average Years at Company

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards
* Conditional Formatting
* Excel Functions

## 📈 Dashboard Features

* Employee Attrition Analysis
* Department-wise Analysis
* Job Role Analysis
* Gender-wise Analysis
* Age Group Analysis
* Education Field Analysis
* Salary Analysis
* Interactive Filters (Slicers)

## 📂 Dataset

This project is based on the **IBM HR Analytics Employee Attrition** dataset.

## 🚀 Outcome

The dashboard provides clear and interactive visualizations that help HR teams analyze employee data, monitor important metrics, and make informed business decisions.
